## [1.0.0] - 2026-03-11

### Added
- Initial stable release of Explorica.

- `data_quality` module for dataset inspection:
  missing value analysis, constant feature detection, categorical feature
  identification, outlier removal, and discretization utilities.

- `interactions` module for statistical relationship analysis:
  Cramér's V, eta-squared, multiple correlation coefficient, and nonlinear
  correlation index.

- `visualizations` module providing plotting utilities for exploratory data analysis.

- `reports` module for generating structured analysis summaries.

- Full Sphinx documentation with API reference and usage examples.
